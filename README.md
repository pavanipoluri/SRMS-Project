# Student Record Management System (C++ | OOP | File Handling)

This project is a Student Record Management System developed using C++ with Object-Oriented Programming (OOP) concepts. It allows users to add, view, search, modify, and delete student records. All data is stored permanently using binary file handling.

## Features
- Add new student record  
- Display all student records (tabular format)  
- Search student by registration number  
- Modify existing student details  
- Delete student record  
- Prevents duplicate registration numbers  
- Stores data permanently using `.dat` binary file  
- Uses OOP principles: classes, objects, encapsulation  

## Technologies Used
- C++  
- OOP  
- File Handling (Binary Files)  
- iostream, fstream, iomanip
